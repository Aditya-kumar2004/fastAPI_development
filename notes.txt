## Useful commands

- Activate virtualenv (cmd):
```bat
myvenv\Scripts\activate.bat
```

- Activate virtualenv (PowerShell):
```powershell
.\myvenv\Scripts\Activate.ps1
```

- Install dependencies:
```bash
pip install fastapi uvicorn[standard]
```

- Run the app with reload:
```bash
uvicorn main:app --reload
```

MONGODB

mongoDB:- It is a physical conatiner for coollection(known as entity in SQL)
coollection:- Group  of document and similar to RDBMStable. A colection exist in one databse only. It doest enforce schema
document:- it is a record that store inside collection and its similar to ows in the table(rdbms) and it is a key-value pair of data
field :- It is a key-value pair of data(known as column in SQL)

**) Incoming data will in json format

Each databse get it own set of files on the filesystem(on the hardisk)::
1.db-1.ns: a ns(namesspace) file that stores metadata about the collections in that database
2.db-*.0 : data files for each collection(data in chunk )
3.db-*.ns  : index files for each collection (in most cases )
4.db-last-rs: rollback segment (only for replication)
