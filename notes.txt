## Useful commands

- Activate virtualenv (cmd):
```bat
myvenv\Scripts\activate.bat
```

- Activate virtualenv (PowerShell):
```powershell
.\myvenv\Scripts\Activate.ps1
```

- Install dependencies:
```bash
pip install fastapi uvicorn[standard]
```

- Run the app with reload:
```bash
uvicorn main:app --reload
```

- Test `/students` endpoint example:
http://127.0.0.1:8000/students?name=Aditya&depart=CSE&cgpa=5.5
